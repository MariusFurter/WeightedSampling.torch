# Contributing

Thanks for your interest in contributing to WeightedSampling.torch!

## Getting Started

1. Fork the repository and clone your fork.
2. Install in development mode:
   ```bash
   pip install -e .
   pip install pytest
   ```
3. Run the tests to make sure everything works:
   ```bash
   pytest tests/ -v
   ```

## Making Changes

- Create a branch for your work.
- Keep changes focused — one feature or fix per pull request.
- Add or update tests for any new functionality.
- Make sure all tests pass before submitting.

## Reporting Issues

If you find a bug or have a feature request, please open an issue on GitHub with a clear description and, if applicable, a minimal reproducing example.
